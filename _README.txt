Tanks Game ~ Getting Started

Uses libgdx, eclipse, and gradle
For more information on LibGDX visit https://libgdx.badlogicgames.com


Current directory is the main eclipse project.
The two sub projects are desktop and core.
Source code can be found in desktop and core folders.
Assets (images and music) are found in core/assets/ folder.

To RUN the game use the _TanksGame.jar file
